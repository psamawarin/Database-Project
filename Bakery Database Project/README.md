Functions list:
    
- `register_customer`: Inserts a new customer into the database.
- `get_login_customer`: Fetches customer credentials for login.
- `record_customer_login`: Updates the last login timestamp upon customer login.
- `check_custom_cake_allergens`: Retrieves allergens based on custom cakes and customer preferences.
- `create_order`: Generates and inserts a new order.
- `get_customer_addresses`: Fetches addresses linked to a specific customer.
- `browse_products_for_customer`: Displays products with allergen conflicts for customers.
- `add_customer_address`: Adds unique customer addresses with validation.
- `add_product_to_order`: Adds products to an existing order.
- `cancel_order`: Cancels an existing order.
- `finalize_order`: Finalizes an order and updates the inventory.
- `get_customer_order_history`: Retrieves order history for a specific customer.
- `get_order_details`: Retrieves detailed information about an order.
